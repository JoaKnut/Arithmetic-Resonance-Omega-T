# Arithmetic-Resonance-Omega-T
Marco de Resonancia Aritmética basado en Omega(n)=d(2n)-4. La función caracteriza la primalidad, propone una formulación de la Conjetura ABC y define constantes fundamentales. El modelo dinámico Psi_E(n) postula una equivalencia física para la Hipótesis de Riemann. Este repositorio incluye la teoría y el código de simulación.
